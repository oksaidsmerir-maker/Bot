# Documentation

## Setup Instructions

1. Clone the repository:  
   ```bash
   git clone https://github.com/oksaidsmerir-maker/Bot.git
   ```

2. Navigate into the directory:  
   ```bash
   cd Bot
   ```

3. Install the required dependencies:  
   ```bash
   npm install
   ```

4. Run the application:  
   ```bash
   npm start
   ```

5. For testing, you can use:  
   ```bash
   npm test
   ```
